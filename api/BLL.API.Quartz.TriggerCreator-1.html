CreateTriggerForever(Nullable<DateTime>, Int32, String, Dictionary<String, Object>)

Создание задачи планировщика, который будет срабатывать каждые N минут (Вроде крона) Если в запланированное время сервер не был способен выполнить задачу (к примеру был отключен) при запуске сервера планировщик НЕ запустит выполнение моментально, а дождется следующего времени в соответствии с интервалом, тем самым мы не теряем график запуска задач
